En programmation, une liste chainée est une série d'objets dans laquelle chaque objet est appelé un nœud. Ces nœuds contiennent des données ainsi que des références ou liens vers le nœud suivant.
Une pile, en programmation, est une structure de données qui suit le principe LIFO(last In first Out).Le dernier élémént à avoir été ajouté est susceptible d'etre le premier à etre retiré. Il est utile pour ce qui est d'annuler une action précédemment accomplie.
La recherche binaire est un type d'algorithme qui opère des divisions successives sur un tableau déja trié. Le but est de trouver un élément en particulier. Si cet élément n'est pas égal à l'élément du milieu du tableau, l'algorithme rejette la partie du tableau dans laquelle l'élément ne peut pas se trouver et se concentre ainsi sur l'autre.
La recherche lineaire est un algorithme très simple dont le but est de parcourir un tableau ou une structure de données quelconque pour trouver un élément spécifique correspondant à une condition pré-établie.
